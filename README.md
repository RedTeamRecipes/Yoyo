# Snowden The Security Search Engine 

**Snowden** is a specialized search engine that indexes **only** vetted, high-value cybersecurity and offensive security resources , General-purpose search engines are flooded with SEO spam, generated content, and beginner-level tutorials. Snowden cuts through the noise. If it's on Snowden, itâ€™s a resource worth reading.

### Key Features

* **Curated Indexing:** We manualy vet or community-verify every domain we crawl.
* **Offensive Security Focus:** Prioritizes CTF writeups, 0-day research, exploit databases, and technical deep-dives.
* **No SEO Spam:** We strictly ban content farms and paywalled generic articles.
* **Community Driven:** The index grows through open-source contributions.


### Criteria for Inclusion
Before submitting, please ensure the resource meets our standards:
1.  **High Technical Value:** Must provide actual value (code, research, distinct methodology).
2.  **No Paywalls:** Content must be accessible (standard blogs, git repos, wikis).
3.  **Originality:** No sites that purely scrape/repost other people's work.

### Submission Methods

#### Method 1: Pull Request (Recommended)
1.  Fork this repository.
2.  Navigate to `sites.txt`.
3.  Add the URL and a category tag (e.g., `Exploit Dev`, `Web Sec`, `Reverse Engineering`,.....).
4.  Submit a Pull Request with the title: `[ADD] Name of Sites`.

#### Method 2: Issue Tracker
Simply open an issue with the tag **"New Source"** and provide the link. Our maintainers will vet it and add it to the crawler.
